# oscp
Personal OSCP scripts

## pwndex
Searches for pocs on Git and SearchSploit. Clone the repos right from the tui!

## recon server
Sets up a server to track your targets, nmap them, track credentials, track flags

It also creates obsidian notes pages for all of the targets and can produce a final report from the notes
